UX / UI DESIGN CASE STUDY

Find The Best Way
to Learn Languages

CanTalk is a scenario-based language learning app designed to bridge the gap between vocabulary memorization and real-life communication.

Try Interactive Prototype

Assignment Context

GBDA 210 - Introduction to UX Design. A comprehensive project focused on solving real-world communication barriers through mobile app design.

My Role

UX Designer & Prototyper. Managed the end-to-end design process, from user research and wireframing to high-fidelity prototyping.

Problem & Objective

Many learners struggle with real-life conversation. CanTalk bridges this gap through scenario-based learning and intuitive community interactions.

Impact & Reflection

The prototype received high praise for smooth navigation. I learned the vital importance of testing low-fi sketches before high-fi visuals.

Scenario Examples

Practice for upcoming interviews and build confidence through simulated conversations.

Discover local events and language exchange meetups to connect with new people.

Outlining core user flows and information architecture to validate features early.
